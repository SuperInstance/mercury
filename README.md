# mercury

**Mercury logic programming** — a modern logic/functional programming language implementation. Part of the SuperInstance OpenConstruct ecosystem.

## What This Gives You

- **Logic programming** — declarative programming with relations and constraints
- **Functional features** — higher-order functions, type inference, pattern matching
- **Constraint solving** — built-in constraint propagation and unification
- **OpenConstruct integration** — compiles to targets supported by the OpenConstruct ABI

## How It Fits

Language tool in the SuperInstance OpenConstruct ecosystem. Provides logic programming capabilities alongside `openconstruct-mercury` bindings. Used for constraint specification in the `conservation-spectral` pipeline.
